# Revard

This project aims to be an Angular 20 application written in TypeScript with
plain CSS. Normally you would use the Angular CLI to create it:

```bash
npm install -g @angular/cli@20
ng new revard --defaults --style=css
```

The CLI cannot be run here, so the repository mirrors the structure the CLI
would generate.

## Getting Started

1. Install dependencies (requires internet access):
   ```bash
   npm install
   ```
2. Serve the application:
   ```bash
   npm start
   ```

Formatting and linting are configured with **Prettier** and **ESLint**. You can run them with:

```bash
npm run format
npm run lint
```
